* Setting Up Locally

1. First clone down the repo, load the schema, and run the seed file.

```

$ git clone https://github.com/davidrf/launchvotes.git
$ rake db:setup

```

2. Start the server.

```

$ rails server
Visit your application.

```
3. Visit your [application](http://localhost:3000/)
